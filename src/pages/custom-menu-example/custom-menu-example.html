Custom Menu Example
Case 01
Side Menu With Custom Top Navbar Menu ( no extras )

As is often the case with a "Admin" style template, one with a left sidebar menu and a top navbar menu. In this case we have the main left menu which will be the main menu on mobile and the top right menu. In order to setup this up you first need to create the menu items in the ./app/app.pwa.config.ts file. 

    public static readonly MenuItemsUnique = [
      { title: 'Account', component: 'HomePage' },
      { title: 'Logout', component: 'CenterExamplePage' },
    ];

Then in your DOM when you declare the pwa-menu add the isUniqueMenu="true" 

    ‹ion-buttons end padding-right›
        ‹pwa-menu isUniqueMenu="true"›‹/pwa-menu›
    ‹/ion-buttons›

Left as is, this unique top right hand menu will disappear on anything smaller then 922px

Case 02
Addition Of Unique Menu In Main Menu On Smaller Screen Sizes

A common senario would be to add the unique top right hand side toolbar menu items into the main left hand side menu when on smaller devices. This is NOT set by default but easy to add if you would like that functionality.

In the ./app/app.component.ts file you need to activate 

      appendUniqueMenuItems: boolean = true;
      hasSetUniqueMenuItems: boolean = false;
This will then trigger the setUniqueMenu() conditionals. Resize the screen with the boolean values added to view the result.
